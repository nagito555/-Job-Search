Бот по поиску работы.

Это телеграм бот с функционалом поиска работы. 
При написание команды " /start" Бот предложит назвать профессию, сменить регион (Изначалльно стоит Ижевск), найти регион по названию и отправить список всех регионов


Имеет следующие команды:

/search_reg - поиск региона по названию
